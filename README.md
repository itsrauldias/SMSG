Este projeto foi escrito em [Next.js](https://nextjs.org/).

## Iniciando o projeto

Primeiramente, inicie o projeto utilizando os comandos:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
